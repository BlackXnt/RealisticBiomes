RealisticBiomes configuration spreadsheet

NEW: All rates are now displayed as with maximum soil bonus for convenience

All values are pulled from config file in the official repo. Percentages are calculated on page load, based on interpretation of the plugin's code. This interpretation may be wrong, open an issue if you find errors.

Notes:

  1. Hover over table headers to get more information.
  2. All times are displayed as if plant had full soil fertilizer, full sunlight etc.
  3. Some biomes are correctly conflated for readability (e.g. "ICE"), reflecting configuration by 06-12-2014. A future configuration change might break current conflation.
  4. Sunlight level means exposure to sky: transparent blocks are taken into account and time of day is irrelevant.
  5. You should be able to figure everything out from this table. For more in depth explanation of RealisticBiomes please check the wiki.
